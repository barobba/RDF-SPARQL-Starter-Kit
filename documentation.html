Current Endpoint
The endpoint is the connection to the graph.
Current Graph
Graphs are used to arbitrarily segment the data.
For example, two applications could have separate graphs for the same type of data.
Namespaces
Namespaces help shorten the data into human readable pieces.
Basic Input Form

For entering triples into the graph.

The three input fields represent subject, predicate, object triples in RDF.

For example:

_ex:book1 dc:title "Little House on the Prairie"

For the object field make sure to add double quotes (") when entering strings or other datatypes which aren't URIs.
