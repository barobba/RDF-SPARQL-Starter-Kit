Sample SPARQL Statements

SPARQL is the query language for RDF.

Select

PREFIX prefix <URI>
SELECT
  *
WHERE {
  GRAPH ?graph {
    ?subject ?predicate ?object .
  }
}

PREFIX prefix <URI>
SELECT
  *
WHERE {
  ?subject ?predicate ?object .
}

Insert

PREFIX prefix <URI>
INSERT INTO <GRAPH> {
  subject predicate object .
}

Update

Use DELETE then UPDATE commands together.

Delete

PREFIX prefix <URI>
DELETE FROM <GRAPH> {
  subject predicate object .
}
